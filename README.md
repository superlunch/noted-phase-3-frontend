# Noted

> A simple note-taking app. Built with Ruby, Sinatra, and React.

<br>

![logo](public/noted-wide-1.png)

<br>


## Features
+ Create an account
+ Create notes & add them to project categories
+ Edit note bodies & titles
+ Delete notes
+ Search notes by project category or contents
